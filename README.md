# Architecture Inspiration

The architecture and project structure of this application were inspired by the following open-source repositories and engineering patterns:

* [Clean Express Architecture](https://github.com/coodos/clean-express)

* [Clean Express Architecture specfic mongodb](https://github.com/LovelySehotra/ExpressMongoDB)
These repositories helped shape the overall backend structure, modular design, scalability approach, error handling strategy, and service-layer organization used in this project.
